# AI Assistance Disclosure

DivApply contains code that was developed with AI assistance and then reviewed and adapted by the repository owner.

The project is intentionally transparent about that history. The AI-assisted workflow was used to help with implementation, refactoring, and documentation, but the repository owner remains responsible for the final code and its behavior.

If you contribute changes, please keep that same standard of review and accountability: AI-assisted work is welcome, but it should be checked carefully before merge or release.
